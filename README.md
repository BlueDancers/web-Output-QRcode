## 介绍

通过**node**读取xls文件，批量生成二维码。

## 基本用法

1. 克隆仓库，下载相关依赖。

2. 打开命令行,执行

   ```js
   // 将xls中所有的数据都转换成二维码
   npm run build
   // 将xls中的一条数据转换成二维码
   npm run test
   ```
